INSTRUCCIONES PARA EJECUTAR EL SISTEMA WEB

1. Abre la consola y ubícate en esta carpeta:
   cd %USERPROFILE%\Downloads\control_calidad_web_complete

2. Instala las dependencias:
   py -m pip install -r requirements.txt

3. Ejecuta la aplicación:
   py app.py

4. Abre tu navegador y ve a:
   http://127.0.0.1:5000

Desde ahí podrás registrar controles de calidad y exportar a Excel.
